# Project Images

Place your project screenshots in this folder to replace the emoji placeholders.

## Recommended Images:

1. **fraud-detection.png** (or .jpg) - Screenshot of your fraud detection project
2. **data-analyzer.png** - Screenshot of DATA ANALYZER platform
3. **finanalyse.png** - Screenshot of FinAnalyse application
4. **mafinance-pro.png** - Screenshot of MaFinance Pro dashboard

## Image Guidelines:
- **Format**: PNG or JPG
- **Recommended size**: 1200x600px or 16:9 aspect ratio
- **File naming**: Use lowercase with hyphens (e.g., fraud-detection.png)
- **Quality**: High resolution for best display
- **Content**: Showcase the key features/UI of your projects

## How to Add Images:

After placing images in this folder, update the HTML:

### Current (with emoji placeholder):
```html
<div class="project-image">
  <div class="project-placeholder">ðŸ”’</div>
</div>
```

### Updated (with your image):
```html
<div class="project-image">
  <img src="images/fraud-detection.png" alt="Fraud Detection Project">
</div>
```

The images will have automatic hover zoom effects and mocha-themed styling!
